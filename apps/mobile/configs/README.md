# Tenant Configs

This directory contains tenant-specific JSON configuration files generated by the build system.

Each tenant gets a JSON file with:
- `tenantId` — Unique tenant identifier
- `appName` — Display name for the app
- `storeUrl` — E-commerce store URL to load in WebView
- `onesignalAppId` — OneSignal app ID for push notifications
- `apiBaseUrl` — Backend API URL
- `theme` — Color scheme (primaryColor, backgroundColor, accentColor)
- `logoUrl` — App logo URL (stored in Cloudflare R2)
- `splashUrl` — Splash screen image URL
- `features` — Feature flags (biometrics, haptics, offlineMode)

## File naming

```
{tenant_id}.config.json
```

## Example

```json
{
  "tenantId": "abc123",
  "appName": "My Store",
  "storeUrl": "https://mystore.myshopify.com",
  "onesignalAppId": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
  "apiBaseUrl": "https://api.appfy.com.br/v1",
  "theme": {
    "primaryColor": "#A855F7",
    "backgroundColor": "#050505",
    "accentColor": "#D946EF"
  },
  "logoUrl": "https://r2.appfy.com.br/abc123/logo.png",
  "splashUrl": "https://r2.appfy.com.br/abc123/splash.png",
  "features": {
    "biometrics": true,
    "haptics": true,
    "offlineMode": false
  }
}
```

These configs are generated by the console's build system and copied into the app at build time.
